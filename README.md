Caching DNS Resolver
====================

## Example

**index.js**
```javascript
async function run() {
  console.log('resolve("zeit.co")');
  for (let i = 0; i < 10; i++) {
    console.time('resolve');
    console.log('IP: ', await dnsResolve('zeit.co'));
    console.timeEnd('resolve');
  }
}
run().catch(console.error);
```
